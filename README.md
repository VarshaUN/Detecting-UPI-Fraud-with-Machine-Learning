# Detecting-UPI-Fraud-with-Machine-Learning-Leveraging-Festive-patterns
India’s UPI processes 150B+ transactions yearly but loses ₹1,200 crore to fraud. My ML model analyzes 100K transactions, uncovering higher fraud during festivals (4% vs. 2%) and achieving 78.7% accuracy with Logistic Regression. This work strengthens digital payment security in India.
